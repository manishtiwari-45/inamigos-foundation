## 📌 About the Project

This webpage was created to raise awareness about InAmigos Foundation and its ongoing
social initiatives. It covers the NGO's mission, all 6 active projects, real impact
statistics, recent events, and multiple calls-to-action for volunteering and donations.

All content is sourced directly from the official website and social media channels.

## 🖼 Images Setup

Images are **not included** in this repository. Download them from the official sources
and place them in the same folder as `index.html`.

| File | Source |
|---|---|
| `logo.png` | [inamigosfoundation.org.in](https://inamigosfoundation.org.in) — top-left logo |
| `hero-bg.jpg` | Instagram [@inamigos](https://instagram.com/inamigos) — event/group photo |
| `about.jpg` | Facebook/Instagram — founder or team photo |
| `project-*.jpg` | Instagram — respective project activity photos |
| `event-*.jpg` | [Events page](https://inamigosfoundation.org.in/events) |
| `gallery-*.jpg` | Instagram [@inamigos](https://instagram.com/inamigos) — any 4 activity photos |

> The page displays styled placeholders when images are missing, so it will not break.

## 🚀 Deployment

This site is deployed via **Netlify with GitHub auto-deploy**.

Every push to the `main` branch automatically triggers a redeployment.

**Workflow:**
```bash
git add .
git commit -m "your update message"
git push
```

**To deploy your own copy:**
1. Fork this repository
2. Go to [netlify.com](https://netlify.com) → New Site → Import from GitHub
3. Select the forked repo
4. Set publish directory to `.`
5. Deploy
